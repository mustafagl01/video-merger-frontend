# Video Merger Frontend

Simple Next.js dashboard for the Video Merger API.

## Local setup

1. Install deps: `npm install`
2. Create env file: copy `.env.local.example` to `.env.local`
3. Start dev server: `npm run dev`

## Environment

`NEXT_PUBLIC_API_URL=http://46.224.148.239:8000`

## Build

`npm run build`
